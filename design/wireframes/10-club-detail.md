# 10 — Club Detail Page

**Route**: `/clubs/[clubId]`  
**Auth state**: any authenticated user (footballer or club viewing another club).  
**Goal**: Public-facing club profile — history, roster, stadium, recent posts, and subscription CTA for footballers.

## Annotations

| ⚑   | Note                                                                                                                                                                |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| A   | Subscribe button only visible to FOOTBALLER users. Shows "გამოწ." (subscribed) / "გამ. გაუქ." (unsubscribed). Toast on toggle.                                      |
| B   | Roster "profile link" column only shows platform profile links for roster entries that are linked to a verified footballer account; free-text entries show nothing. |
| C   | Tab routing: `/clubs/[id]`, `/clubs/[id]?tab=roster`, `?tab=stadium`, `?tab=news`. Default is history/bio.                                                          |
| D   | Club detail is publicly visible (no auth required for the page itself), but subscribe CTA redirects to `/auth/signin` if unauthenticated.                           |
| E   | News tab items link to individual post pages `/clubs/[id]/posts/[postId]` (Phase 7).                                                                                |
| F   | Mobile uses horizontally scrollable tab strip. Cover photo is full-width at top.                                                                                    |
