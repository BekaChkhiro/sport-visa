# 03 — Onboarding Wizard

**Routes**: `/onboarding` (role-aware; renders footballer or club steps)  
**Auth state**: authenticated, profile incomplete.  
**Goal**: Collect the minimum required profile data before first dashboard access.

## Annotations

| ⚑   | Note                                                                                                                      |
| --- | ------------------------------------------------------------------------------------------------------------------------- |
| A   | Positions are displayed with abbreviated labels; hover shows full name (e.g. GK = მეკარე). Multi-select allowed (max 2).  |
| B   | Video links are validated client-side with a YouTube/Vimeo URL regex; embed preview shown inline after paste.             |
| C   | Admin verification is manual for MVP. User can still browse the platform but cannot be contacted by clubs until verified. |
| D   | Skipping optional fields is allowed; a "complete later" nudge shown on the dashboard.                                     |
